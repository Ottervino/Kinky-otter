## PROMPT CHEAT SHEET: Copy & Paste This Every Time

### The Style Stack (ALWAYS use this first)
```
Bold vector illustration, thick black outlines, flat solid colors,
no gradients, minimal shading, Tom of Finland influence,
clean silhouette, high contrast shapes, vector graphic quality.
```

### Add Your Subject in the Middle
```
Character: [describe who they are]
Build: [lean / stocky / muscular / average]
Facial hair: [beard / stubble / clean]
Clothing: [harness / leather vest / jockstrap / boots / cap]
Pose: [standing / seated / hands on hips / facing viewer]
Expression: [confident / playful / dominant / warm]
Mood: [leather bar / pride / biker / bear community]
```

### Always End With
```
Output: clean white background, die-cut sticker border,
centered composition, vector ready illustration.
```

### Full Example (Copy/Paste Ready)
```
Bold vector illustration, thick black outlines, flat solid colors,
no gradients, minimal shading, Tom of Finland influence,
clean silhouette, high contrast shapes, vector graphic quality.
Character: lean muscular otter-type man, 30s, body hair on chest and arms,

Output: clean white background, die-cut sticker border, centered composition.
```
